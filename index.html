Анжелика Вакулич
Капитальные ремонты генератора проводились в 2008, 2016 годах. Текущие ремонты проводятся ежегодно. Техническое состояние генератора удовлетворительное, замечания по работе отсутствуют. Выемка ротора производилась при капитальных ремонтах персоналом филиала «Белоозёрскэнергоремонт» РУП «Брестэнерго» через специальный технологический проём в стене здания с выводом ротора на площадку на крыше КРУ-6 кВ. Ротор выводится практически целиком, в «расточке» статора остаётся только часть полумуфты, что не критично при ремонте. Проект по выводу ротора из «расточки» статора присутствует, изготовлен ОАО «Привод-Сервис». Необходимая для выемки ротора оснастка имеется в электрическом цеху филиала.
